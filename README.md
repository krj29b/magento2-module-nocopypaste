<p align="center"><b>Webverse_NoCopyPaste</b></p> 
<p>A Magento 2 module that disables the following behaviour on your storefront: CTRL+C, CTRL+V, CTRL+SHIFT+I (inspect), CTRL+U (page source) Right Click. </p>
